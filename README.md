## git log


Привет! сегодня я познакомлю тебя с особенностью этой команды.


Команда git log используется для отображения истории коммитов в репозитории.  Если после выполнения этой команды вы не можете выйти из отображаемого журнала,  это происходит потому, что git log по умолчанию использует пейджер (например, less),  который позволяет прокручивать и просматривать длинный список коммитов.
